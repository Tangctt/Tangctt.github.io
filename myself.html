    # Stories

    # 足球生涯⚽

    本科四年期间，每天不是去踢球，就是在去踢球的路上，也因此结识了一些“狐朋狗友”。可如今，随着一年接一年的毕业季，大家都已分道扬镳，有的去工作，有的读研，还有的报效祖国。

    花有重开日，人无再少年。曾经一起踢球的绿茵场换了一波又一波的同学，我们存在过的痕迹也渐渐消失。九院的球场没了我们依旧很热闹，这片土地也在无时无刻上演着我们的故事。

    • 谨以此纪念那些年一起踢球、一起欢乐、一起陪我度过我大学足球时光的兄弟们

    # 拉萨之旅🚲

    一场说走就走的旅行，源于对骑行的热爱，对自由的向往，和对远方的幻想。

    G318-成都=>拉萨，总里程2155KM，历时22天，翻越12座海拔4000+的雪山...

    一路上虽历经风雨、高反等诸多困难，可还是敌不过世界最美景观大道带给我们的视觉冲击和心灵震撼。

    禾尼的草原、然乌的阳光、米堆的冰川、拉萨的夕阳，是去过一次就一定想去第二次的地方，现在依然清晰地印在我的脑海之中。

    当看到布达拉宫，一切皆释然，故事也随之戛然而止...

    • 怀念那段梦幻的时光、自由骑行的我们，和令人流连忘返的拉萨之旅
