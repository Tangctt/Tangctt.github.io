    # ✨Node.js基础✨

    # var xxx = require('模块')

    1. 模块指核心模块：http、fs、os等
    2. 第三方模块：express（用于搭建web服务）
    3. 自定义模块

    # 作用

    1. 执行被加载模块代码
    2. 得到被加载模块中的exports导出接口对象

    # 如何创建一个HTTP服务

    1. 创建server
    2. 监听request请求事件，设置请求处理函数
    3. 绑定端口号，重启服务

    # 服务器

    • 服务器就是一个24消失不关机的计算机
    • ip地址用来定位计算机
    • 端口号用来定位具体的应用程序
    • 所有需要联网通信的应用程序都会占用一个端口号
